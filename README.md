# Ski Race Betting (VS Code Extension)

Local ski race tipping app inside VS Code.

## Run the extension (development)

1. Install dependencies:

```bash
npm install
```

2. Start the Extension Development Host:
- In VS Code, open this folder
- Press **F5** (or use **Run and Debug** → **Run Extension**)

3. Use it:
- In the new VS Code window that opens, look for the **SkiRace** icon in the left Activity Bar
- Use the actions: Add ski race / Add player / Add tipp / Add result / Show Tipp results

## Persistence

Data is stored locally on the same PC using VS Code's extension global storage (a JSON file under the extension storage directory).
